## Technologies
Project is created with:
* React Native version: 0.70.5
* React version: 18.1.0
* Expo cli version: 6.0.5

## Installation
### For Android
        - yarn install
        - npm install
        - expo start --android
### For IOS
        - npm install
        - cd ios
        - pod install
        - npx expo start --ios
### For link fonts
        - npx react-native-asset
